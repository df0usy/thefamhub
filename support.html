TheFamHub Support

Need help with TheFamHub? Use the information below for common support topics and contact.

Support Contact

Email: support@thefamhub.app

If you do not own this address yet, replace it before submitting to the App Store.

Common Issues

  • If family location is not updating, confirm location permission is set to Always.
  • If notifications are not arriving, confirm notifications are enabled both in iOS Settings and in the Family notification preferences.
  • If Apple Calendar sync is unavailable, confirm calendar access has been granted in iOS Settings.
  • If an invite link does not work, make sure the invited person opens it on a device where TheFamHub is installed and signs in first if required.

Account and Data Requests

If you need help with account access, family membership, or privacy-related questions, contact support using the address above.
